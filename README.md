# JP IT Staffing Website

This is a modern, fully responsive website I built for JP IT Staffing as part of my internship project.

## What I Built

A professional recruitment website with these features:
- Responsive design that works on all devices
- Dark/Light mode toggle
- Interactive animations and smooth scrolling
- Working contact form
- Clean, modern UI

## Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript
- **Backend**: PHP (for contact form)
- **Database**: MySQL (optional)
- **Libraries**: Particles.js, AOS (Animate On Scroll), Font Awesome
