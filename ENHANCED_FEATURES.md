## ⚙️ Customization:

### Change Colors:
Open `css/style.css` and modify:
```css
:root {
    --primary-color: #667eea;     /* Your color */
    --secondary-color: #764ba2;   /* Your color */
}
```

### Toggle Features:
In `js/script.js`:
```javascript
// Disable particles
// Comment out: particlesJS('particles-js', {...});

// Disable cursor trail
// Comment out the cursor trail section
```

### Adjust Animation Speed:
In `js/script.js`:
```javascript
AOS.init({
    duration: 1000,  // Change this (ms)
    easing: 'ease-in-out',
    once: true
});
```
